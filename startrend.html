STARTREND — Twin Spike

Version: v1.01 • Engine: Twin-Spike Detection + RetroScan

What it does

STARTREND patiently identifies a first strong spike then waits for a validating second spike in the same direction before entering. It includes deep retro-scan, spike-decay invalidation, and configurable risk settings.

Why traders use it

  • Market-structure based — avoids noise
  • Very selective — fewer, higher-quality trades
  • Robust to indicator smoothing via re-validation

Quick Specs

Default timeframe: H1 • Default SpikeThreshold: 11 • SL 50 pips • TP 100 pips
